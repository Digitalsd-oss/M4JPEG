Download Page

 

The M4JPEG tool is a Matlab-compiled program, but it doesn't require the Matlab IDE to be installed. The running of a Matlab-compiled program just requires the installation of Matlab Compiler Runtime (MCR). The MCR enables the user to run applications compiled within Matlab. It does not require a Matlab license and can be used to run the Matlab-compiled program on computers that do not have Matlab installed.
Downloads for M4JPEG ver 1.40 / Win 32 bit platform

There are two M4JPEG installers to choose from; the first one has the MCR included while the second one needs to download it from the web:

Options Package MCR Web Size
